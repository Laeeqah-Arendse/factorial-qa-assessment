QA Assessment Submission - Factorial Calculator

Author: Laeeqah Arendse

Contents:

* Automation Scripts

Automation:

* Framework: Playwright
* Language: JavaScript

How to Run:

1. Install dependencies:
   npm install
   npx playwright install

2. Run tests:
   npx playwright test

3. View report:
   npx playwright show-report

Notes:

* Some tests are expected to fail due to identified defects.
* Failures validate issues documented in the defect report.
* Additional tests include API validation and UI validation checks.
* Tests were executed across Chromium, Firefox, and WebKit to ensure cross-browser compatibility. Failures were consistent across browsers for functional defects.
